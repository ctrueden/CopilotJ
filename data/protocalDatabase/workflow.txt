ID: workflow_3d_analysis_pipeline
Title: 3D Fluorescence Stack Analysis Pipeline
Tags: 3D Viewer, batch, segmentation, tracking, quantification

--- Steps ---
1. Open all 3D fluorescence stacks in Fiji (File → Import → Image Sequence or drag‐and‐drop).
2. Run the Brightest Point 3D projection on each stack.
   Plugin Reference: macro_3d_project_brightest
3. Adjust brightness/contrast on the projected images to [0,255] and apply to all.
   Plugin Reference: macro_batch_brightness_adjust
4. Perform 3D to 2D segmentation using CellPose (via DeepImageJ plugin).
   Plugin Reference: macro_cellpose_segmentation_multichannel
5. Use TrackMate to track particles in the segmented 2D projections.
   Plugin Reference: macro_trackmate_particle_tracking
6. Export tracking results to CSV and perform quantification in Python or Excel.

--- Notes ---
• Each plugin_reference should match an id in the plugin JSON so RAG can look up the detailed macro syntax automatically.
• Steps without a plugin_reference are manual instructions or post‐processing steps outside of ImageJ.

ID: workflow_simple_segmentation
Title: Single‐Image Segmentation and Measurement
Tags: segmentation, measurement, CellPose, macro

--- Steps ---
1. Open the image in Fiji.
2. Run CellPose segmentation macro on channel 1.
   Plugin Reference: macro_cellpose_segmentation_channel1
3. Measure area and intensity: Analyze → Measure (Alt+M).

--- Notes ---
• This simple workflow shows how a user can combine a plugin macro with native Fiji commands.
